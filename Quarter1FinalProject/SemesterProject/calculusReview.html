Strategies for Finding Limits
simply substitute that one method dividing out technique rationalizing technique Assume we are trying to find the limit of some function f(x) as x approaches c. First, if f is some function, say a polynomial, which allows for direct substitution, then simply plug c in for x. This only won't work when c is an exception on f, like when f is piecewise and c is on a boundry. If this is the case then find the the one-sided limits as x approaches c from the left and right and compare them. If they turn out to be equal, then the limit is their value. If they are different, then the limit fails to exist.
